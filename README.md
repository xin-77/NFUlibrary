# NFULibrary

#### 介绍
软件工程实训项目，基于SpringBoot+Vue实现的智慧图书系统。

#### 软件架构
SpringBoot + Mybatis-Plus + Mysql + Vue + Element Plus


#### 安装教程

1.  运行sql文件创建数据库
2.  配置数据库连接信息
3.  解压前端文件
4.  输入下面指令
    ```
    npm install
    npm run serve
    ```
